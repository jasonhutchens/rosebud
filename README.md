ROSEBUD
=======

Extract web page content.

Basic HTML
----------

* `h1` and `h2`
* `p` and `a` and `pre` and `code`
* `b` and `i`
* `ul` and `ol` and `li`
* `figure` with `img` and `caption`
* `quote`
* `embed`
